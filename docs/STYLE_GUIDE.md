# Visual Style Guide — The Terrible Turtle Oracle

**Chosen direction: Mythic Woodcut + Gold.** All 48 cards must read as one deck. This guide is the shared contract; every card's `image_prompt` prepends the Prompt Preamble below.

## Core aesthetic
- **Medium look:** hand-carved woodblock / letterpress relief print. Bold confident black linework, visible carving marks and chatter, cross-hatch and stipple for shading. No photographic depth, no gradients, no 3D render — flat printed ink.
- **Palette (disciplined):** black ink + a single **metallic gold** accent, printed on warm **desert-tan / kraft paper** (~#D9C4A3, with paper grain). Gold used sparingly for the sacred/axis element and fine details.
- **Per-realm undertone** (one muted spot tint + treatment, so the four realms are subconsciously distinct but the deck stays unified):
  - **Shell** — gold-forward; radiant sunburst / axis motif; most ornament.
  - **Roots** — deep indigo undertone; downward pull; dense subterranean cross-hatching.
  - **Trunk** — rust/ochre undertone; a grounded horizon line; balanced, upright.
  - **Branches** — pale sky-blue undertone; upward reach; airy negative space, small stars.
- **Line weight & lighting:** consistent medium-bold outline; flat, even, print-like light; strong figure-ground.

## Fixed frame (identical on every card, for cohesion)
- **Portrait tarot ratio** (~2:3; render 1024×1536 or similar).
- **Thin double gold border** just inside the card edge.
- **Bottom cartouche/banner** holding the card NAME in a woodcut serif (small caps).
- **Top-corner realm glyph:** 🐢 Shell · roots-knot · trunk-ring · branch-star (rendered as a small carved mark, not emoji).
- **Recurring motif:** the World Tree axis (a vertical line/tree/turtle-shell arc) subtly present in every composition — the spine of the deck.
- Centered, iconic, symmetrical/heraldic subject. Leave calm space so the printed text (Reading + Dare go on the physical card back or lower third) stays legible.

## Prompt Preamble (prepend to each card's image_prompt)
> "Hand-carved woodblock print / letterpress relief illustration, bold black ink linework with visible carving texture, cross-hatch shading, printed on warm desert-tan kraft paper with subtle grain, sparing metallic gold accents, flat print (no 3D, no photo, no gradient), mythic and heraldic, centered symmetrical composition, thin double gold border, a subtle vertical World-Tree axis motif, bottom banner cartouche with the title in a woodcut serif. [REALM undertone: <shell gold / roots indigo / trunk rust / branches sky-blue>]. Subject: "

## Consistency checklist (per generated card)
1. Same paper tone & grain. 2. Same border + title cartouche. 3. Only black + gold + the realm undertone. 4. Woodcut texture present (no smooth render). 5. Axis motif somewhere. 6. Subject centered with breathing room. 7. Realm glyph in the corner.

## Cover / box
Likely **The World Turtle** or **The Resonant Spire** as the box art (the spire literally turns voices into light — the oracle as sculpture). Full-gold treatment.
